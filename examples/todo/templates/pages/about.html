About

A minimal todo app built with Hull — pure HTML forms, server-side rendering with the Hull template engine, and SQLite for storage.

No client-side JavaScript. Every action is a form POST with a redirect back.

Stack

  • Backend: Hull (Lua)
  • Templates: hull.template with inheritance and includes
  • Database: SQLite (via Hull db API)
  • Frontend: Pure HTML + CSS, zero JS

Source

See examples/todo/ in the Hull repository.
